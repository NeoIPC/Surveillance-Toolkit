# List of Organisation Units

The country-level organisation units here — the `NEOIPC` project root and one unit per country — were
generated from the eurostat NUTS history at <https://ec.europa.eu/eurostat/web/nuts/history> (country codes
follow eurostat conventions, e.g. `EL` for Greece and `UK` for the United Kingdom).

The NUTS 1 regions derived from the same source now live with the `play` variant
(`metadata/play/organisationUnits.csv`), alongside the synthetic test hospitals and departments.
